# Scalatra

This repo is created for practicing and understanding 
[Scalatra](http://www.scalatra.org/).

### Build & Run ###

```sh
$ cd WebApp
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open 
[http://localhost:8080/](http://localhost:8080/) in your browser.
